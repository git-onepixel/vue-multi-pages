# vuebone

## Manifest

``` bash
├── build                       # 构建配置
├── config                      # 配置
├── dist                        # 发布  
├── public
│   └── favicon.ico             # Favicon
├── src 
│   ├── assets                  # 静态资源
│   │   ├── fonts       
│   │   └── img    
│   ├── common                  # 公用配置
│   ├── components              # 通用组件
│   ├── pages                   # 业务组件页面
│   ├── styles                  # 通用样式库
│   │   ├── base.less
│   │   ├── vars.less       
│   │   └── mixin.less        
│   └── utils                   # 工具类       
├── package.json 
├── README.md 

```

## Build
``` bash
npm install
# watch:
npm run dev
# build:
npm run build
# lint & fix:
npm run lint
```
